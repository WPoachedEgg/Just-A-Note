## GroupBy

Find Unique item in the array of int.  [LC136](https://leetcode.com/problems/single-number/)

```
nums.GroupBy(x => x)
        .Select(x => new { number = x.Key, count = x.Count() })
        .Where(x => x.count == 1)
        .FirstOrDefault()
        .number;
```
